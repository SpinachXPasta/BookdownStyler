If you are using library or require in your rmd please use the process below to expedite speed up the code run process and to install packages if they are not installed.


req_pkgs = c("purrr","dplyr","tidyr","stringr","ggplot2") # your packages can replace this
installed_pkges = row.names(installed.packages())
d_pk = setdiff(req_pkgs, installed_pkges)
if (length(d_pk) > 0){
install.packages( d_pk, repos = "http://cran.us.r-project.org")
}
